Nintendo DS


Double écran

Avec deux écrans LCD rétro-éclairés d'une très grande définition, la Nintendo DS offre une nouvelle approche du jeu. Les possibilités sont illimitées: l'un des écrans peut servir à montrer l'action principale tandis que l'autre peut être utilisé pour montrer les cartes, inventaires, ou point de vue secondaires, mais les deux écrans peuvent être utilisés simultanément pour montrer les immenses boss !

Innovation

L'écran inférieur réagit au toucher fournissant ainsi une expérience de jeu résolument concrète. Vous pouvez contrôler l'action à l'écran avec plus de précision qu'avant ou naviguer instantanément à travers les menus à l'aide du stylet.
La console présente deux ports de jeu distincts, l'un est destinés aux cartes de jeu DS et l'autre aux cartouches de jeu Game Boy Advance. La console est compatible avec l'intégralité des jeux GBA, ce qui permet à la console de disposer d'un catalogue de jeu impressionnant.
Chaque console est dotée d'un microphone permettant, dans certains jeux, de communiquer avec les autres joueurs mais également d'interagir dans les jeux vidéos de milles manières.
